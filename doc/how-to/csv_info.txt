CSVファイルカラム説明

1. Enabled
    - 説明: プロセスが有効かどうかを示します。
    - 値: "True" または "False"
    - 例: True

2. Process Name
    - 説明: プロセスの名前を示します。
    - 値: 任意の文字列
    - 例: ExampleProcess

3. Executable Path
    - 説明: 実行可能ファイルのパスを示します。
    - 値: ファイルのフルパス
    - 例: C:\path\to\executable.exe

4. Arguments
    - 説明: 実行可能ファイルに渡す引数を示します。
    - 値: スペース区切りの引数文字列
    - 例: arg1 arg2

5. Last Run Time
    - 説明: プロセスが最後に実行された日時を示します。
    - 値: "YYYY-MM-DD HH:MM:SS" 形式の文字列
    - 例: 2024-07-10 12:00:00

6. Frequency
    - 説明: プロセスを実行する頻度を分単位で示します。
    - 値: 正の整数
    - 例: 5
